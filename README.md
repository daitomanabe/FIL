# FIL

This repository contains the source code for the FIL project, comprising an OpenFrameworks application and a macOS Screensaver.

## Structure

- **fil_of_app**: OpenFrameworks application source code.
- **fil_screensaver**: macOS Screensaver source code.

## Requirements

- **fil_of_app**: Requires [OpenFrameworks](https://openframeworks.cc/).
- **fil_screensaver**: Requires Xcode for macOS development.
