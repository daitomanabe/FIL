# fil_screensaver

A macOS Screensaver utilizing minimal threading animations.

## Installation
1.  Download the installer from [Releases](https://github.com/daitomanabe/FIL/releases).
2.  Run `fil_screensaver.pkg`.
3.  Open System Settings -> Screen Saver and select "fil".

## Build from Source
Open `fil_screensaver.xcodeproj` in Xcode and build.
